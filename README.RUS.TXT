jsFUnit - javascript фреймворк для написания функциональных и юнит-тестов.
Версия: 2.3.0

Лицензии:
1. jsFUnit распространяется по лицензии BSD. См. licenses

Документация в каталоге "documentation", а тесты в каталоге "tests-js".

Использование:
1. Распакуйте jsFUnit.zip в каталог с вашим хостингом
1. Создайте тест в tests-js
2. Запустите update.pl:
    $ update.pl
    update.pl обновит Tests.js
3. Откройте тест в http:/ваш_хост/tests или file://localhost/путь/jsFUnit-x.x.x/index.html
