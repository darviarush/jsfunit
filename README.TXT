jsFUnit - javascript functional and unit testing framework.
Version: 2.3.0

Licenses:
1. JsFUnit have BSD license. This license look in directory "licenses".

The documentation in the directory "documentation" and exaples in directory "tests-js".

Using:
1. Extract jsFUnit.zip in your hosting directory
2. Create file with extract ".js" in tests-js
3. Run update.pl:
    $ update.pl
    update.pl update Tests.js
4. Open test in http:/your.host/tests or file://localhost/your.path/jsFUnit-x.x.x/index.html
